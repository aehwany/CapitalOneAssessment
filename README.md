# CapitalOneAssessment
Software Engineering Technical Assessment for CapitalOne new grad role

## To run the code on a specific file
* Add the file path as an input argument to the java program.

## To test the code on sample files
* In each test file, make sure to modify the 'fileName' to the path of the sample file on disk.
* Each test class corresonds to the expected values from its corresponding provided sample file.
* The provided sample files could be found in SampleFiles folder in the root directory of this repository.
